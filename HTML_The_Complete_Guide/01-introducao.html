Introdução ao HTML

HTML é a linguagem usada para estruturar o conteúdo das páginas web. Com ele, podemos criar títulos, parágrafos, imagens, links, tabelas e formulários.

O que significa HTML?

HTML significa HyperText Markup Language, ou Linguagem de Marcação de Hipertexto.

Para que serve?

Ele serve para organizar o conteúdo de uma página. O estilo visual normalmente é feito com CSS e a interatividade com JavaScript.
