# Guia Definitivo de HTML

### Autor: Professor Luciano Rocha

---

## Sobre o projeto

Este repositório contém um **guia completo de HTML**, desenvolvido com o objetivo de servir como material de apoio para alunos de cursos técnicos e iniciantes em desenvolvimento web.

O conteúdo foi organizado de forma **progressiva**, iniciando pelos conceitos básicos e avançando até tópicos mais estruturados do HTML5.

---

## Objetivo

* Facilitar o aprendizado de HTML de forma clara e prática
* Servir como material de consulta para alunos
* Ajudar no desenvolvimento de páginas web desde o zero
* Preparar a base para estudos em CSS e JavaScript

---

## O que você vai aprender

* Estrutura básica do HTML
* Tags de texto e formatação
* Links e navegação
* Inserção de imagens
* Listas e tabelas
* Criação de formulários
* HTML semântico
* Recursos do HTML5
* Boas práticas de desenvolvimento

## Como utilizar

1. Faça o download ou clone este repositório:

```bash
git clone https://github.com/luciano1/aulas/HTML_The_Complete_Guide.git
```

2. Abra o arquivo `index.html` em seu navegador

3. Navegue pelos conteúdos na ordem sugerida

---

## Requisitos

* Navegador web (Chrome, Edge, Firefox)
* Editor de código (recomendado: VS Code)

---

## Dicas importantes

* Sempre utilize:

```html
<meta charset="UTF-8">
```

para evitar problemas com acentuação.

* Salve seus arquivos em **UTF-8**

* Mantenha o código organizado e indentado
