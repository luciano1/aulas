PROJETO MODELO - CADASTRO DE ANIMAIS

Objetivo:
Servir como modelo simples para o trabalho final dos alunos.

O projeto utiliza:
- React com Vite
- 3 componentes principais
- useState
- props
- array de objetos
- map()
- formulário simples
- CSS básico

Componentes:
1. Header.jsx
2. FormularioAnimal.jsx
3. ListaAnimais.jsx

Campos cadastrados:
- Animal
- Raça
- Nome

Como executar:
1. npm install
2. npm run dev
